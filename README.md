# Desafio1
Desafío 1 Alura Latam G9 DC
Este código corresponde al desafío 1 del curso de Data Science de Alura Latam , en el programa Oracle One.
Se solicitó realizar consultas y extracción de datos desde una base de datos en formato CSV. Todas las consultas fueron realizadas, utilizando PANDAS en Python.

Propósito del análisis realizado:

Se realizó un análisis de la base de datos proporcionada, con el fin de que el cliente pueda tomar una decisión informada respecto de la venta de una de sus cuatro sucursales.

El proyecto se realizó en Google Colaboratory y fue copiado a Github.

Se generaron gráficos circular, de barras y lineas para respaldar visualmente la información contenida en las respuestas a las consultas.

Para poder ejecutar o corregir el notebook, se debe abrir en Google Colaboratory (colab.google.com) y guardar una copia.
